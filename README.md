# Cognitive-Fog Analyzer

A simple Python-based tool to help track cognitive fog levels by recording:
- Focus
- Sleepiness
- Understanding
- Productivity

The app stores results in a CSV file and analyzes patterns over time.

## Features
- Simple Tkinter GUI
- Data saved in CSV
- Automatic date & time logging
- Easy to use interface

## How to Run
1. Install Python 3.13.7
2. Download the project
   https://github.com/jmuskan2005-ai/Cognitive-Fog-Analyzer.git
3. Open the Project Folder
   go to the folder where your .py file is saved.
4. Run the Script
    Open your terminal and run:
    python finalprovityarthi.py
5. Use the App
   - The application window will open
   - Adjust the sliders
   - Click Check Fog Level
   - Your Fog Score and category will appear instantly
6. View Saved Data
    Every test is stored automatically in:
     fog_report.csv

## Requirements
   your system should have:
    python 3.13.7
    Tkinter (comes pre-installed with Python)
